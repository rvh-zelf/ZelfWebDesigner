# Scene templates (by output type)

Map user intent → default **frame size** and **density** before designing.

| Scene | Typical frame | Notes |
|-------|---------------|-------|
| Social cover | 900×383 or platform spec | Big type, one hero, safe margins |
| PPT data slide | 1920×1080 | Chart-first; title + one insight |
| Vertical infographic | 1080×1920 | Scroll or paginated sections |
| Marketing site hero | 1440×900+ | Narrative arc above fold |
| AI product landing | 1280×800+ | Show real UI chrome when possible |
| Developer docs | wide readable column | Code blocks + side nav |
| App onboarding | 390×844 per screen | Use device frame starter |

Cross-link: `slide-decks.md` for deck grammar; `app-prototype.md` for phone shells.
