# Design philosophy lanes (abbreviated)

Use as a **menu**, not a rigid taxonomy. Each slot: **name / studio vibe**, one-line hook, prompt nugget.

## Lane A · Information architecture (01–04)

1. **Pentagram-style information architecture** — ruthless hierarchy, editorial grid; "grid + typographic voice, few colours".
2. **Swiss / International** — modular grid, objective photography, Akzidenz-like restraint.
3. **NASA-poster clarity** — instructional contrast, oversized numerals, functional colour coding.
4. **Financial Times / newsprint gravitas** — column rhythm, serif authority, muted ink.

## Lane B · Kinetic / immersive (05–08)

5. **Field.io motion poetry** — fluid camera, depth parallax, soft tech humanism.
6. **Apple keynote kinetic** — slow dolly, light caustics, hero object worship.
7. **Music-visualiser energy** — beat-synced pulses (use sparingly with accessible motion prefs).
8. **XR HUD futurism** — diegetic UI, glass layers, telemetry — only when brief fits.

## Lane C · Minimal / refined (09–12)

9. **Kenya Hara emptiness** — tactile paper, breathing space, single accent earth tone.
10. **Muji system calm** — SKU-like clarity, quiet neutrals, no ornament without cause.
11. **Jil Sander reduction** — couture negative space, precision seams as metaphor.
12. **Dieter Rams honesty** — "less but better", colour only for state.

## Lane D · Experimental (13–16)

13. **Stefan Sagmeister play** — hand type, provocation, tactile collage.
14. **Generative / shader field** — rules-based chaos bounded by grid snaps.
15. **Brutalist web raw** — monospace, exposed structure, high contrast.
16. **Memphis revival** — flat primaries, pattern clash — only when audience matches.

## Lane E · Eastern / poetic (17–20)

17. **Song dynasty scroll pacing** — horizontal narrative drift, ink wash negative space.
18. **Zen garden asymmetry** — odd counts, imperfect balance on purpose.
19. **Neon-night market** — dense vernacular signage systems (use with restraint).
20. **Contemporary CN editorial** — red accent discipline, grid + calligraphy contrast.

**Rule**: pick **three directions from three different lanes** for advisor mode.

**Prompt template**

`[Lane hook] + [content] + [medium: 1920×1080 deck | 390×844 app | 1200×900 web] + [colours with HEX from brand-spec or advisor choice] + [forbidden: purple SaaS gradient unless brand]`
